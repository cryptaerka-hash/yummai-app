📋 Project Brief — YummAI Restaurant Platform

Кто мы

  • Основатель: Erka Maratov, Нукус, Узбекистан (Каракалпакстан)
  • Background: Co-founder FLY TAXI (4+ лет, ridesharing в Нукусе), CS-degree TATU
  • Параллельный проект: SPOT — самая технологичная транспортная платформа Центральной Азии (в разработке с iTechnolabs)
  • Миссия: строить технологические продукты мирового уровня для рынка Центральной Азии

Что такое YummAI

  • SaaS-платформа для автоматизации ресторанов — QR-меню, AI-апселл, кухонный дисплей, аналитика
  • Целевой рынок: рестораны Узбекистана (Ташкент, Самарканд, Нукус) — первая фаза
  • Далее: Казахстан, Кыргызстан, Таджикистан
  • Аналоги: Toast (США), iiko (РФ) — но адаптированные под локальный рынок с AI-слоем

Ключевые модули (MVP)

  • QR-меню: PWA без установки, фото блюд, категории, описания, аллергены
  • Заказ со стола: корзина, модификаторы, комментарии к заказу
  • AI Upsell Engine: рекомендации на основе истории заказов и ассоциативных правил
  • Kitchen Display System (KDS): WebSocket real-time, статусы NEW → COOKING → READY → SERVED
  • Оплата: Click API, Payme API, наличные с фиксацией
  • Аналитика: выручка по дням/блюдам/столам, средний чек, AI-инсайты
  • Панель владельца: управление меню, столами, ценами, акциями
  • R-Keeper интеграция: JSON-RPC bridge для ресторанов с существующей кассой

Технический стек (предложенный)

  • Backend: NestJS + TypeScript, PostgreSQL + Prisma ORM, Redis (кеш + pub/sub)
  • Frontend: React + Tailwind CSS + Zustand (state)
  • Real-time: WebSocket (Socket.IO) для KDS и live-обновлений
  • PWA: Workbox + IndexedDB (offline-first для гостевого меню)
  • Payments: Click API + Payme API (нативные UZS интеграции)
  • SMS: Eskiz.uz или Playmobile (верификация, чеки)
  • AI: GPT-4 API или локальная ML-модель для recommendation engine
  • Infra: Docker, CI/CD, облако (AWS/DigitalOcean/локальный хостинг)

Бизнес-модель

  • SaaS подписка: 1 500 000 — 2 500 000 UZS/месяц (≈ $120–$200)
  • Окупаемость клиента: 1 месяц (рост чека +23% перекрывает стоимость)
  • Target Year 1: 50 ресторанов → 75M–125M UZS/месяц MRR
  • Уникальное преимущество: первый AI-powered ресторанный SaaS в Узбекистане

Что нужно от iTechnolabs

  • Разработка MVP: 3-4 месяца, полный цикл (дизайн → бэкенд → фронт → деплой)
  • Мобильная версия (PWA) + веб-панель для владельца
  • Интеграции: Click, Payme, R-Keeper, Eskiz.uz
  • AI-модуль: recommendation engine (можно старт с простыми ассоциативными правилами)
  • Документация и передача кода основателю
  • Post-launch поддержка 3 месяца
