Project Brief — YummAI Restaurant Platform

Кто мы

  • Основатель: Erka Maratov, Нукус, Узбекистан (Каракалпакстан)
  • Co-founder FLY TAXI (4+ лет, ridesharing в Нукусе), CS-degree TATU
  • Параллельный проект: SPOT — транспортная платформа Центральной Азии
  • Миссия: технологические продукты мирового уровня для рынка Центральной Азии

Что такое YummAI

  • SaaS-платформа автоматизации ресторанов: QR-меню, AI-рекомендации, кухонный экран, аналитика
  • Целевой рынок: Узбекистан (Ташкент, Самарканд, Нукус) — первая фаза
  • Далее: Казахстан, Кыргызстан, Таджикистан
  • Аналоги: Toast (США), iiko (РФ) — но с локальными платежами и AI-слоем

Ключевые модули MVP

  • QR-меню: PWA без установки, фото блюд, категории, аллергены
  • Заказ со стола: корзина, модификаторы, комментарии
  • AI Upsell Engine: ассоциативные правила + ML на истории заказов
  • Kitchen Display System: WebSocket real-time, статусы NEW → COOKING → READY → SERVED
  • Оплата: Click API, Payme API, наличные с фиксацией
  • Аналитика: выручка по дням / блюдам / столам, AI-инсайты
  • Панель владельца: меню, столы, цены, акции
  • R-Keeper интеграция: JSON-RPC bridge для существующих касс

Технический стек

  • Backend: NestJS + TypeScript, PostgreSQL + Prisma, Redis
  • Frontend: React + Tailwind CSS + Zustand
  • Real-time: Socket.IO (WebSocket)
  • PWA: Workbox + IndexedDB (offline-first)
  • Payments: Click API + Payme API
  • SMS: Eskiz.uz / Playmobile
  • AI: GPT-4 API или собственная recommendation engine
  • Infra: Docker, CI/CD

Бизнес-модель

  • SaaS подписка: 1 500 000 — 2 500 000 UZS/месяц (≈ $120–$200)
  • Окупаемость клиента: 1 месяц
  • Target Year 1: 50 ресторанов → 75M–125M UZS/месяц MRR
  • Первый AI-powered ресторанный SaaS в Узбекистане

Что нужно от команды разработки

  • MVP за 3–4 месяца: дизайн → бэкенд → фронт → деплой
  • PWA + веб-панель для владельца
  • Интеграции: Click, Payme, R-Keeper, Eskiz.uz
  • AI-модуль рекомендаций (старт с простыми ассоциативными правилами)
  • Документация и передача кода
  • Поддержка 3 месяца после запуска
